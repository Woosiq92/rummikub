# 🃏 루미큐브 AI 대전 연습 게임

인공지능과 대결하면서 루미큐브 게임을 연습할 수 있는 웹 기반 게임 플랫폼입니다.

## ✨ 주요 기능

### 🎮 게임 플레이
- **드래그 앤 드롭**: 타일을 드래그하여 테이블에 배치
- **조합 검증**: 세트(Set)와 런(Run) 자동 검증
- **턴 관리**: 플레이어와 AI의 턴 교대
- **게임 규칙**: 첫 플레이 30점 이상, 조커 사용 규칙 등

### 🤖 AI 플레이어
- **4가지 난이도**: 쉬움, 보통, 어려움, 전문가
- **난이도별 전략**: 각 난이도에 맞는 AI 플레이 스타일
- **실수 확률**: 난이도별 실수 확률 조절

### 📊 통계 시스템
- **게임 통계**: 총 게임 수, 승률, 평균 점수, 최고 점수
- **난이도별 통계**: 각 난이도별 상세 통계
- **게임 기록**: 최근 100개 게임 기록 저장

### 💾 저장 기능
- **게임 저장**: 진행 중인 게임 저장
- **게임 불러오기**: 저장된 게임 재개
- **로컬 스토리지**: 브라우저 로컬 스토리지 사용

### 📖 도움말
- **게임 규칙**: 상세한 게임 규칙 설명
- **조작 방법**: 드래그 앤 드롭, 버튼 사용법
- **난이도 설명**: 각 난이도별 특징 설명

## 🎯 게임 규칙

### 조합 규칙
- **세트(Set)**: 같은 숫자, 다른 색상 (최소 3개)
- **런(Run)**: 연속 숫자, 같은 색상 (최소 3개)

### 게임 진행
1. 각 플레이어는 14개의 타일을 받습니다
2. 첫 플레이는 30점 이상의 조합이 필요합니다 (조커 사용 불가)
3. 타일을 드래그하여 테이블에 배치합니다
4. 모든 조합이 유효해야 플레이할 수 있습니다
5. 모든 타일을 사용하면 승리합니다

## 🚀 시작하기

### 설치 및 실행
1. 프로젝트 폴더를 다운로드합니다
2. `index.html` 파일을 브라우저에서 엽니다
3. 난이도를 선택하고 게임을 시작합니다

### 브라우저 지원
- Chrome (권장)
- Firefox
- Safari
- Edge

## 🎮 난이도 설명

### 🌱 쉬움 (Easy)
- 기본 조합만 사용
- 실수 확률: 20%
- 초보자용

### ⭐ 보통 (Normal)
- 최적 플레이
- 실수 확률: 10%
- 일반 플레이어용

### 🔥 어려움 (Hard)
- 전략적 플레이
- 테이블 타일 활용
- 실수 확률: 5%
- 고수용

### 👑 전문가 (Expert)
- 최적의 전략
- 테이블 타일 활용
- 실수 확률: 1%
- 마스터용

## 🔧 기술 스택
